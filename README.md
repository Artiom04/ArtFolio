# ArtFolio
## 📖Introduzione
Questa è la Repo di ArtFolio,  progetto di Ingegneria del Software dell'A.A 2025-2026.

ArtFolio è un sistema digitale pensato per le istituzioni AFAM (Alta Formazione Artistica, Musicale e Coreutica) con l’obiettivo di permettere agli studenti di costruire e gestire la propria identità digitale in modo moderno, strutturato e controllato.

## 📂Repository
La repo è organizzata per facilitare la navigazione tra le varie sezioni. Ecco la struttura principale:
```
┌── Codice/               # ⚙️ I codici delle parti applicative del progetto
│   ├── Boundary
│   ├── Control
│   ├── Entity
│   ├── Properties
│   ├── Resources
│   ├── Program.cs
│   ├── WinFormApp2.csproj
│   └── WinFormsApp2.sln
├── Documenti/             # 📄 I documenti che sono stati prodotti
│   ├── RAD.pdf
│   ├── SDD.pdf
│   └── ODD.pdf
├── Tesina 25-26.pdf	    # 🎓 Indicazioni per lo sviluppo del progetto
└── README.md              	# ℹ️ Il file che stai leggendo ora
```
