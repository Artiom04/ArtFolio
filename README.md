# ArtFolio
## 📖Introduzione
Questa è la Repo di ArtFolio,  progetto di Ingegneria del Software dell'A.A 2025-2026.

ArtFolio è un sistema digitale pensato per le istituzioni AFAM (Alta Formazione Artistica, Musicale e Coreutica) con l’obiettivo di permettere agli studenti di costruire e gestire la propria identità digitale in modo moderno, strutturato e controllato.
